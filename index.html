Undocumented ETL Codebase Reverse-Engineering

40,000 Lines Decoded in Under 1 Day · Solo Investigator · 2025

Inherited a 40+ file Python ETL codebase with zero documentation — 22 transaction types, ~7,000 column mappings, 90 staging tables to 115 silver tables. New team members were taking 2-4 weeks to become productive.

Developed a structured 3-pass AI prompt strategy: Pass 1 — skeleton extraction (file roles, entry points, dependency graph). Accepted 85-90% automation accuracy with manual verification on edge cases versus 100% manual review at 10x the time.

MOSAIC AI Data Assistant

Solo Architect & Developer · 2025 - 2026

A 200+ table Lloyd's insurance platform had no AI-assisted investigation capability. Every data quality incident meant 4-6 hours of manual SQL scripting. With 8+ investigations per month, the team was spending weeks on reactive analysis that could have been hours.

I designed a multi-layer AI system: a custom Python MCP server exposing 10 sandboxed read-only tools over live Redshift, integrated with GitHub Copilot SDK and a Chainlit conversational UI. Coforge · XFI-to-Redshift Pipeline

40,000-Line ETL Codebase Decoded in Under a Day

Solo Investigator · 2025

Inherited a 40+ file Python ETL codebase with zero documentation. 22 transaction types, ~7,000 column mappings, 90 staging tables feeding 115 silver tables. New engineers were taking 2-4 weeks to become productive — a recurring cost no one had quantified.

Developed a structured 3-pass AI prompt strategy: Pass 1 extracted the skeleton (file roles, entry points, dependency graph). Pass 2 mapped every column transformation, business rule, and SCD Type 2 pattern. Pass 3 validated extractions against live Redshift data to catch hallucinations. Accepted 85-90% automation accuracy with targeted manual verification rather than 100% manual review at 10x the time.
